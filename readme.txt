Alle Programming files sind hier zu finden :)

Die Files NEW_multiple_fullVSpectrums_Titan und Plot_p_Eff haben die richtigen V-Spektren implimentiert.

Um ein V-Spektrum zu erstellen einfach NEW_multiple_fullVSpectrums_Titan öffnen die Nummer an Files pro run eingeben und plot_runs_vSpect (ohne frac) mit den runs die man haben will aufrufen.
Die Analyse für Titan Guides wird automatisch nur auf die Runs mit Titan Guides angewandt.


Im Ordner des Programms müssen die Daten der Runs unter pf2data/DAQ/run_10/FILTERED/DataF_run_10 angelegt sein.
